Projeto: GoDices - Aplicação para rolagens de dados [JAVA]

Status do projeto: Finalizado

Uma aplicação para a rolagem de dado de RPG do sistema de Ordem Paranormal.

Ele funciona apenas para ataques de armas/itens que causam dano. Exemplo, a aplicação pode fazer um teste automático de uma katana,
podendo colocar o teste, modificadores de teste, dano causado, tipo de dano, margem de crítico com cálculo de crítico automático
e amostragem de dano e dano calculado.

Conteúdo:

  • Rolagem de dados.
  • Pastas independentes para os dados.
  • Separação de dano.
  • Cálculo automático de testes e danos.
  • Apenas console.
  • Alguns pequenos bugs.
